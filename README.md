**Day101 - 生成AIで作るセキュリティツール200**

**Color Hack Box** は、色の三要素（色相・明度・彩度）をインタラクティブに操作しながら学べる教育ツールです。

スライダーを動かすだけで、HSL（色相・彩度・明度）が色の見え方にどのように影響するかを直感的に理解できます。

HEX・RGB・HSL形式でのカラーコード表示により、デザイン実務にも活用できます。

---

## 💡 デザインはセンスではなく知識

「デザインセンスがないから…」と諦めていませんか？

**配色は才能ではなく、学べる技術です。** 色相環、補色関係、トーン理論、配色比率──これらはすべて体系化された知識であり、誰でも習得できます。

このツールは、色彩理論を「感覚」ではなく「論理」として学ぶために設計されています。スライダーを動かし、数値を確認し、規則性を発見する。その繰り返しが、確かな配色スキルを育てます。

**勉強すれば、誰でも美しい配色ができるようになります。**

## 🎯 目的

以下の目的で開発しています。

- **色彩の基礎教育**: 色相・明度・彩度の関係を視覚的に理解
- **直感的な学習体験**: スライダー操作による即座のフィードバック
- **実務への応用**: HEX/RGB/HSL形式での色指定方法を習得
- **デザイン補助**: 配色検討時のクイックリファレンスとして活用

## 📚 活用事例

### 1. デザイン教育での色彩理論指導
専門学校やデザインスクールにおいて、色の三要素を教える際の実習教材として活用できます。
従来の教科書による説明だけでなく、スライダーを操作することで、色相・明度・彩度の変化が色の印象にどう影響するかを体感的に理解できます。
特に「彩度を0にすると無彩色になる」「明度を変えると同じ色相でもまったく違う印象になる」といった概念を、視覚的に即座に確認できる点が教育効果を高めます。

### 2. Webサイト・アプリUI設計での配色検討
Webデザイナーやフロントエンド開発者が、サイトやアプリの配色を決める際のリファレンスツールとして使用できます。
例えば、ブランドカラーをベースに、明度や彩度を調整して派生色を作成したり、補色関係を確認したりする作業をスムーズに行えます。
HEX・RGB・HSL形式でカラーコードが表示されるため、そのままCSSやデザインツールに転用できる実用性も備えています。

### 3. アクセシビリティ検証のための色調整
WCAGガイドラインに準拠したWebサイト制作において、テキストと背景のコントラスト比を確保する際に活用できます。
明度スライダーを調整することで、視認性の高い配色を探索し、高齢者や視覚障害者にも読みやすいデザインを検討できます。
また、彩度を下げた状態での見え方を確認することで、色覚多様性（色弱・色盲）への配慮も可能になります。

## 🎨 機能

### 📑 7つのタブ構成

本ツールは7つのタブで色彩理論を体系的に学べます（全44個のアコーディオン形式の解説）：

#### 1️⃣ タブ1: 基礎（色の三要素）
- **インタラクティブスライダー**: 色相・明度・彩度をリアルタイム調整
- **カラーコード表示**: HEX・RGB・HSL形式で即座に変換
- **ワンクリックコピー**: カラーコードをクリップボードにコピー
- **ランダム生成・リセット**: 色の探索と初期化が簡単
- **詳細解説**: 各属性の役割と使い方を6つのアコーディオンで説明

#### 2️⃣ タブ2: 無彩色とグレースケール
- **グレースケールスライダー**: 黒〜白まで無彩色を生成
- **プリセットボタン**: 純白・ライトグレー・ミドルグレー・ダークグレー・純黒を瞬時に設定
- **実用的な解説**: 無彩色の役割、コントラスト比、HEXコード、モノクロデザインの実例

#### 3️⃣ タブ3: 色の印象と心理
- **色別心理効果**: 赤・青・緑・黄・オレンジ・紫・ピンク・黒白の9色の心理的影響を解説
- **連想タグ**: ポジティブ・ニュートラル・ネガティブの印象を視覚化
- **色のバリエーション**: 各色の明度・彩度を変えた実例を表示
- **活用シーン**: 業種・目的別の配色アドバイス

#### 4️⃣ タブ4: トーン別パレット
- **トーンマップ**: 明度×彩度の9つのトーン（ペール、ビビッド、ダル、ディープなど）を視覚化
- **トーン別カラーグリッド**: 各トーンの色見本を一覧表示
- **配色テクニック**: 同一トーン配色、グラデーション配色の実例

#### 5️⃣ タブ5: 配色と比率
- **70:25:5の法則**: ベース・メイン・アクセントカラーの黄金比
- **60:30:10のパターン**: バランスの良い3色配色
- **視覚的な比率バー**: 実際の面積比を体感
- **実例デモ**: Webサイト風の配色レイアウト

#### 6️⃣ タブ6: 配色例・目的別パレット
- **50+の配色パレット**: コーポレート、EC、ブログ、ポートフォリオなど目的別
- **業種別パレット**: 医療、教育、飲食、IT、金融、エンターテイメント
- **季節別パレット**: 春夏秋冬の代表的な配色
- **雰囲気別パレット**: 信頼感、エネルギッシュ、やさしい、高級感など8種類
- **HEXコード表示**: すぐに実務で使えるカラーコード

#### 7️⃣ タブ7: 雑学・補足
- **色の文化的な違い**: 赤・白・黄・緑が文化圏で持つ異なる意味
- **色覚異常への配慮**: ユニバーサルデザインの実践ポイント
- **RGB vs CMYK**: 画面と印刷の色表現の違い
- **Webセーフカラー**: 歴史と現代における意義
- **色温度**: 暖色・寒色の心理的・生理的効果
- **Webカラーキーワード**: CSSで使える色名140+種類
- **色の錯覚効果**: 同時対比・面積効果・進出色と後退色
- **ガンマ補正**: ディスプレイの特性と実務への影響

### 🎯 インタラクティブ機能

- **補色の色相環**: ホバーで補色ペアをハイライト表示（タブ1）
- **類似色相の色相環**: 隣接30度の類似色をインタラクティブに表示（タブ1）
- **トースト通知**: コピー成功時のフィードバック
- **レスポンシブデザイン**: モバイル（600px）・タブレット（768px）・デスクトップ完全対応

### 🛠️ 技術仕様
- **色変換アルゴリズム**: HSL→RGB→HEX変換を実装（script.js:92-125）
- **アコーディオンUI**: 44個の折りたたみ可能な解説セクション
- **依存関係ゼロ**: 純粋なHTML/CSS/JavaScript（ビルド不要）
- **ファイルサイズ**: HTML 197KB、CSS 58KB、JS 14KB

## 🚀 使い方

### セットアップ

依存関係やビルドツールは不要です。最新のWebブラウザーがあればすぐに使えます。

1. **リポジトリをクローン**:
```bash
git clone https://github.com/ipusiron/color-hack-box.git
cd color-hack-box
```

2. **ブラウザーで開く**:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

または、ローカルサーバーで起動:
```bash
# Python 3の場合
python -m http.server 8000

# Node.js（http-serverがインストール済みの場合）
npx http-server
```

その後、`http://localhost:8000` にアクセス

### 基本的な使い方

1. **スライダーを調整**: Hue（色相）、Lightness（明度）、Saturation（彩度）のスライダーを動かす
2. **色を確認**: カラーチップでリアルタイムに色の変化を確認
3. **コードを取得**: HEX/RGB/HSL形式のカラーコードをコピーして使用
4. **ランダム生成**: 「ランダム生成」ボタンで新しい色を探索
5. **リセット**: 「リセット」ボタンで初期状態に戻す
6. **学習**: 「色の三属性とは？」を展開して色彩理論を学ぶ
